Kalender (källkod)

Prenumenerar man på en kalender i google kalender så kan man varken redigera eller ta bort händelser eftersom att man inte äger kalendern man prenumererar på själv. Detta medför att kalendern blir överfull av händelser som man inte kan ta bort eller dölja.

För att komma runt detta problem har jag skrivit ett program som med hjälp av googles kalender-API kopierar alla mina kalendrar och prenumerationer till en ny kalender där jag faktiskt kan ändra och ta bort alla händelser.

Tyvärr är inte googles API dokumentation den bästa och jag lade ner projektet på grund av buggar och andra problem.

SMS (källkod)

Google kalender stödde sms notifikationer av händelser till Augusti 2015. Innan dess kunde man genom kalender-APIet skapa händelser med sluttid om 1 minut och på så sett får ett sms när händelsen började med valfri text. På det viset kunde min server tala om när den gick ner, startade upp, osv..
